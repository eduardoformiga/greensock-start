GreenSock

Set behavior from INITIAL STATE to FINAL STATE with animations for all items with delay between them

      
        TweenMax.fromTo(items, {
          scale: .3,
          rotation: 180,
          duration: 1,
          stagger: 0.5
        }, {
          scale: 1.1,
          rotation: 0,
          duration: 1,
          stagger: 0.5
        });
